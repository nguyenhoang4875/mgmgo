    We have some usability standards for our website.

    Button

    We have some common standards for buttons:

    • - Positive buttons (OK, Agree, Continue, Yes, Submit, Save,...) have green color (success color in bootstrap) and on the left side.

    • - Negative buttons (No, Cancel, Abort,...) have white color and on the right side.

    • - Destructive buttons (Delete, Discard, Reject, Block,...) have red color (danger color in bootstrap) and on the left side.

    • - Information buttons (Learn more, See detail,...) have blue color and on the left or the middle side.

    Tables

    With table, we have some standards:

    • - No vertical border

    • - horizontal border 1px

    Messages

    With message, we have some standards:

    • - Error message messages have red color and red background-color (danger color in bootstrap) and displayed below the field where the error occurs

    • - Success message messages have green color and green background-color (success color in bootstrap)

    • - Warning message messages have yellow color and yellow background-color (warning color in bootstrap)

    Search form

    With search form, we have some standards:

    • - Button search has only icon search and on the right side.

    • - Search input has placeholder (Ex: Search for places,...) and on the left side

    Form

    With form, we have some standards:

    • - Label is always above of input.

    • - Text input has placeholder (Ex: Enter your name.)

    • - Submit button is always at the end of form
